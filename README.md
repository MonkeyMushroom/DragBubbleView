# DragBubbleView

高仿qq消息气泡拖拽，用到贝赛尔曲线。

支持气泡状态监听回调，状态包括拖拽、移动、复原、消失。

支持的属性有气泡半径、颜色，消息个数文本、字体颜色、字体大小。

效果如下

![image](https://github.com/MonkeyMushroom/DragBubbleView/raw/master/2.gif)

关于如何让控件不拘泥于本身大小，只需在控件父布局添加 android:clipChildren = false 即可
